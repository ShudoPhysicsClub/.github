# 修道物理班 - Shudo Physics Club

広島県広島市中区千田町に位置する修道中学校・修道高等学校の班(部)活動の一つ。<br>
文化班の中では班員数トップを誇る。<br>

## 主な活動内容
- プログラミングによるアプリケーション・ゲーム開発
- LEGOを用いたロボットの研究
- チャット型AIを用いたAI技術の活用
- 電子工作による作品制作
- アマチュア無線による無線交信
- 大会に向けたe-sportsの練習 (中学2年生以上から)
